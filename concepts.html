Key Concepts in Web History

  1. World Wide Web: A network of linked documents accessible through the internet.
  2. Hyperlink: A clickable reference to another web resource.
  3. Web Browser: Software that displays web pages (e.g., Mosaic, Netscape, Chrome).
  4. HTML: The markup language used to create web pages.
  5. CSS: The styling language that controls layout and design.
  6. JavaScript: Adds interactivity and dynamic features to web pages.
  7. Internet: A vast computer network linking smaller computer networks worldwide.
  8. Network: A set of computers which are connected to each other and operate as part of the same system, able to exchange information and messages.
  9. Local Area Network: Two or more connected computers that share resources in a certain location.
  10. Wide Area Network: Two or more computers or LANS located apart from each other that are connected together for the purposes of sharing information
  11. Server: A computer on a network that runs specialized software and stores information in order to "serve" it to the other networked computers.
  12. Client: A computer that runs a web browser or email software and communicates with the server to get the information it needs.
  13. Uniform Resource Locator: A web page's unique address on the World Wide Web that tells the browser exactly where to find the web pages.
  14. Graphical User Interface: Allows the user to navigate a browser or program by pointing and clicking with a mouse instead of typing.
  15. Search Engine: A specialized website that allows you to search through the billions of websites on the World Wide Web to find exactly what you want..

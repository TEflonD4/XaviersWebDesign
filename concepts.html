Key Concepts in Web History

  1. World Wide Web: A network of linked documents accessible through the internet.
  2. Hyperlink: A clickable reference to another web resource.
  3. Web Browser: Software that displays web pages (e.g., Mosaic, Netscape, Chrome).
  4. HTML: The markup language used to create web pages.
  5. CSS: The styling language that controls layout and design.
  6. JavaScript: Adds interactivity and dynamic features to web pages.
